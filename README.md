# AI Fundamentals

인공지능 및 머신러닝의 기본 개념을 다루는 Jupyter 노트북 모음입니다. 이 노트북들은 파이썬 기초부터 AI의 고급 주제까지 학습 경로를 제공하도록 구성되어 있습니다.

## 디렉토리 구조

*   **/00-Python-Core-Progamming**: 데이터 구조, 제어 흐름, 함수, 파일 입출력 및 클래스를 포함한 파이썬 기초.
*   **/01-Foundations-with-NumPy**: 수치 계산을 위한 NumPy 소개.
*   **/02-Machine-Learning-Basics**: 경사 하강법, 선형 회귀 및 k-NN과 같은 기본 머신러닝 알고리즘.
*   **/03-Search-and-Optimization**: 언덕 오르기, 담금질 기법 및 유전 알고리즘과 같은 검색 및 최적화 알고리즘.
